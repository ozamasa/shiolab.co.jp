# Shiojiri Data Story

塩尻市の公開統計データをもとに、人口・年齢構成・教育・産業・財政・暮らしの変化を、ストーリーとして読める形に可視化するプロジェクトです。

## 現在の構成

1. 暮らしの単位はどう変わった？
2. 年齢のバランスはどう変わった？
3. 子どもと学校はどう変わった？
4. 働く場はどう変わった？
5. 行政は何にお金を使うようになった？
6. 暮らしの基盤はどう変わった？

## 主なデータ

- 人口・世帯
- 年齢3区分
- 小学校・中学校の児童生徒数
- 事業所数・従業者数
- 財政規模
- 目的別歳出
- 自動車保有台数

## 主な追加機能

### ストーリー型UI

カテゴリ別にグラフを並べるのではなく、STEPごとに「問い」「読み解き」「結論」を置く構成にしています。

### 目的別歳出

一般会計歳出当初予算額の目的別データを追加し、民生費・教育費・土木費・公債費などの構成比と金額を見られるようにしています。

### 人口構造と財政の接続

以下の補助グラフを追加しています。

- 高齢者人口と民生費を重ねて見る
- 子ども人口と教育費を重ねて見る

人口側は主に国勢調査ベースのため5年刻み、財政側は毎年データです。表示上は欠損年をまたいで線を接続しています。

### イベント線・市長任期帯

主要イベントは縦線、市長任期は背景帯として表示しています。

市長任期帯は時期を読むための補助情報であり、政策との因果関係を示すものではありません。

## 主な設定ファイル

- `public/data/story_steps.json`  
  STEPごとのタイトル、リード文、結論、ナビ表示名を管理します。

- `public/data/sections.json`  
  各グラフの設定、使用データ、軸、注釈、表示順を管理します。

- `public/data/timeline_events.json`  
  グラフ上に表示する主要イベントを管理します。

- `public/data/mayors.json`  
  市長任期帯を管理します。

- `config/datasets.yml`  
  取得・生成するデータセットの設定を管理します。

## データ生成

目的別歳出データを生成する場合:

```bash
python scripts/build.py --dataset finance_expenditure_purpose --force-download
```

全体を再生成する場合:

```bash
python scripts/build.py --force-download
```

ローカル表示:

```bash
python -m http.server 8000 -d public
```

ブラウザで以下を開きます。

```text
http://localhost:8000/
```
