学習を「続く形」にする

SHIOLAB は、学習・教育をテーマにした Web サービスを提供しています。 「毎日少しずつ」「間違えたところをすぐ復習」「迷わず学習に入れる」。 そんな学習体験を大切にしています。

学習コンテンツの中心は、英検対策の無料クイズサービス SHIOLAB QUIZ に掲載しています。

クイズを開く 英検4級 英検3級

英検学習のポイント（基礎）

英検4級・3級では、単語暗記だけでなく、 「短い英文を正しく読む」「会話の流れを理解する」力が重要です。

  • 主語＋動詞の基本語順を意識する
  • よく出る動詞（go / have / like など）を文で覚える
  • Yes / No で答えられる質問文に慣れる

SHIOLAB QUIZ では、これらを穴埋め形式で繰り返し練習できます。

4級は「be動詞／一般動詞／疑問文（Do you〜?）／前置詞（in/on/at）」の取りこぼしが合否に直結します。
3級は「接続詞（because/so/but）」「比較」「不定詞（to + 動詞）」など、文の形が少し複雑になる点がポイントです。

提供中のコンテンツ

英検対策クイズ

英検4級・3級・準2級の頻出テーマを中心に、1回あたり最大25問をランダム出題します。 回答後すぐに解説が表示されるため、テンポよく復習できます。
